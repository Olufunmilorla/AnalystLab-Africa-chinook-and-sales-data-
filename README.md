# SQL Business Analysis Projects

## Overview

This repository contains two SQL-based business analysis projects completed as part of the AnalystLab Africa Data Analytics Internship Program. The projects demonstrate the application of SQL for data exploration, business intelligence, customer analytics, revenue analysis, and performance optimization using real-world datasets.

The objective of these projects was to strengthen practical SQL skills while solving business problems through structured querying and data analysis.

---

# Project 1: Chinook Music Store Analysis

## Project Description

The Chinook Database is a digital music store database containing information about customers, invoices, tracks, artists, albums, and sales transactions.

This project focused on analyzing sales performance, customer purchasing behavior, artist revenue contribution, and overall business performance using SQL queries.

## Business Questions Answered

* Which country generates the highest revenue?
* Who are the top spending customers?
* Which tracks are purchased most frequently?
* Which artist generates the highest revenue?
* What is the average order value?
* How many invoices and customers exist within the database?

## Key Insights

* Total Revenue: $2,328.60
* Total Customers: 59
* Total Invoices: 412
* Highest Revenue Country: USA
* Highest Revenue Artist: Iron Maiden ($138.60)
* Best Selling Tracks: Dazed and Confused and The Trooper
* Average Order Value: $5.65

## SQL Concepts Applied

* SELECT, WHERE, ORDER BY
* GROUP BY and HAVING
* Aggregate Functions (SUM, AVG, COUNT)
* INNER JOIN and LEFT JOIN
* Subqueries
* Window Functions
* Customer Revenue Analysis

# Project 2: Sales Data Analysis (AdventureWorks Lite Equivalent)

## Project Description

This project involved analyzing a sales dataset containing customer transactions, product information, territories, and revenue data. The objective was to identify key revenue drivers, evaluate product performance, understand customer purchasing behavior, and analyze sales trends.

The project utilized both core and advanced SQL concepts to transform raw transactional data into actionable business insights.

## Business Questions Answered

* Which country generates the highest revenue?
* Which product lines contribute the most sales?
* Who are the highest-value customers?
* What are the monthly and quarterly revenue trends?
* How does customer purchasing behavior impact revenue?
* Which products generate the highest revenue?

## Key Insights

* Total Revenue: $10,032,628.85
* Total Orders: 307
* Highest Revenue Country: USA ($3,627,982.83)
* Best Performing Product Line: Classic Cars ($3,919,615.66)
* Top Customer: Euro Shopping Channel ($912,294.11)
* Highest Revenue Month: November 2004 ($1,089,048.01)

## SQL Concepts Applied

* WHERE
* ORDER BY
* GROUP BY
* HAVING

### Aggregate Functions

* SUM()
* AVG()
* COUNT()
* MIN()
* MAX()

### Advanced SQL

* INNER JOIN
* LEFT JOIN
* RIGHT JOIN
* Subqueries
* ROW_NUMBER()
* RANK()
* PARTITION BY

### Query Optimization

* Index Creation
* Query Performance Evaluation
* Statistics Monitoring
